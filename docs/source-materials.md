## 3. 不同素材分别补什么

这次蒸馏不是把所有材料混成一锅，而是给它们分工：

| 素材类型 | 主要补什么 |
|---|---|
| 官方资料 | 事实锚点 |
| 年度演讲 | 长期价值观、公开表达主线 |
| 深度访谈 | 决策逻辑、方法论细节 |
| 视频字幕 / 评论 | 表达节奏、现场感、公众锚点 |
| 微博样本 | 近期风格、短句形态、账号语境 |
| 长文本书籍 | 跨阶段重复主题、长期结构验证 |

所以这个 skill 的蒸馏，不是单一来源，而是一个多源证据拼图。

---

## 4. 哪些素材没有公开分发

这点必须讲清楚。

公开仓库不会原样打包这些内容：

- 长文本书籍原文
- 内部证据层的原始长料
- 仅用于补强验证的未清洗输入

公开仓库分发的是：
- skill 本体
- 结构化 references
- 蒸馏方式说明
- 素材清单说明
- 打包好的 `.skill` 与 zip

也就是说：

**公开版交付的是“蒸馏结果”，不是“原料仓库”。**

共同构成的一套多源证据池。
