## 三、这些素材分别解决什么问题

不同素材解决的问题不同：

- **官方资料**：锁定事实锚点
- **演讲**：看长期价值观与公开表达主线
- **深访**：看方法论与判断过程
- **视频字幕**：看表达节奏与现场口吻
- **微博样本**：看近期表达与短文风格
- **书籍输入**：看跨多年、跨阶段的稳定主题是否成立

## 四、没有直接公开分发的内容

为了保留边界，这个公开 skill 仓库 **不会分发长文本书籍原文**。

原因有两个：

1. 这些材料更适合作为内部蒸馏输入，而不是公开市场附带资产
2. 公开仓库应该分发的是“蒸馏后的结果”，不是原始长文本本身

## 五、一句话总结

这个 skill 用到的素材不是单一路径，而是：

**官方资料 + 演讲 + 深访 + 视频 + 微博样本 + 本机长文本书籍**

共同构成的一套多源证据池。
